# prj02

## Components:
- stepper: Two Trees 17HS4401-22B (NEMA17, 6L-4W, 12V/1.5A, 2-phase, 0.14Nm)
- driver: unknown A4988 StepStick clone (Allegro A4988)
- MCU: Teensy 3.1
- PSU: Takasago ZX-400H (up to 0-640V/0-5A, max 400W), preset to 15V/3.0A
